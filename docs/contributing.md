# 贡献指南

> 欢迎为 Kunnix 做出贡献！

## 🎯 贡献方式

### 1. 提交 Issue

- 🐛 **Bug 报告** - 发现问题？告诉我们
- 💡 **功能建议** - 有新想法？欢迎提出
- 📚 **文档改进** - 发现文档错误？帮忙修正

访问：[GitHub Issues](https://github.com/kunnix-ai/memory-task-system/issues)

### 2. 提交 PR

**流程**:
1. Fork 仓库
2. 创建分支 `git checkout -b feature/your-feature`
3. 提交更改 `git commit -m "Add your feature"`
4. 推送到分支 `git push origin feature/your-feature`
5. 提交 Pull Request

### 3. 完善文档

- 改进现有文档
- 添加使用示例
- 翻译文档（中英文）

### 4. 分享案例

- 分享你的使用经验
- 撰写博客文章
- 录制视频教程

## 📋 代码规范

### Python 代码

- 使用 Black 格式化：`black kunnix/`
- 遵循 PEP 8 规范
- 添加类型注解
- 编写单元测试

### 文档规范

- 使用 Markdown 格式
- 代码示例要完整可运行
- 添加必要的注释

## 🔧 本地开发

```bash
# 1. Fork 并克隆
git clone https://github.com/your-username/memory-task-system.git
cd memory-task-system

# 2. 安装依赖
pip install -e .
pip install -r requirements-dev.txt

# 3. 运行测试
pytest

# 4. 格式化代码
black kunnix/
```
